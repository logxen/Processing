Name

textFont()
Examples 
PFont metaBold;
// The font "Meta-Bold.mvlw" must be located in the 
// current sketch's "data" directory to load successfully
metaBold = loadFont("Meta-Bold.mvlw");
textFont(metaBold); 
text("word", 15, 50);

Description Sets the current font. The font must be loaded with loadFont() before it can be used. This font will be used in all subsequent calls to the text() function.
Syntax 
textFont(font)
Parameters
font   PFont: any variable of the type PFont
Returns None

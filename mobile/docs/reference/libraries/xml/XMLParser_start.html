XMLParser class
Name

start()
Description This method will start parsing an XML document in the background. The results will be returned in the libraryEvent callback.
Syntax 
parser.start(document);
parser.start(is);
Parameters
parser   XMLParser: any variable of type XMLParser
document   String: string containing an XML document
is   InputStream: an open InputStream positioned at the beginning of an XML document
Returns None

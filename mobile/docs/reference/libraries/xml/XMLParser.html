XML library
Name

XMLParser
Description This class is used to parse an XML document. Two methods of parsing are supported, one that runs in the background while the sketch continues to run through its loop, and one that will finish parsing completely before returning control to the sketch.
Methods
parse()   This method will parse an XML document completely before returning the results as an XMLElement.
start()   This method begin parsing an XML document in the background.
stop()   This method will stop an XML document being parsed in the background.
attribute()   This method is used to retrieve the value of an attribute during background parsing.
EVENT_TAG_START   This event occurs when a new tag has been encountered while parsing an XML document in the background.
EVENT_TEXT   This event occurs when text between a set of tags has been parsed.
EVENT_TAG_END   This event occurs when an closing tag has been encountered while parsing an XML document in the background.
EVENT_DOCUMENT_END   This event occurs when an XML document being parsed in the background is finished.
Constructors 
XMLParser(parent)
Parameters
parent   PMIDlet: typically use "this"

Sound class
Name

stop()
Description Stops playback of this sound. This will release the sound hardware so that another sound can be played. As a result, it will take longer to start the sound if you play() it again. If you wish to temporarily pause a sound, use pause() instead.
Syntax 
sound.stop()
Parameters
sound   Sound: any variable of type Sound
Returns None

Bluetooth class
Name

EVENT_CLIENT_CONNECTED
Examples 
Bluetooth bt;

void setup() {
  bt = new Bluetooth(this);
  bt.start("My service");
  noLoop();
}

void libraryEvent(Object library, int event, Object data) {
  if (library == bt) {
    switch (event) {
      case Bluetooth.EVENT_CLIENT_CONNECTED:
        Client client = (Client) data;
        client.writeUTF("Saying hello!");
        client.close();
        break;
    }
  }
}

Description This constant value is reported in libraryEvent when when a client has connected to the Bluetooth service started by this sketch. The data is a Client object that can be used to communicate with the client.
Syntax 
Bluetooth.EVENT_CLIENT_CONNECTED

Bluetooth library
Name

Bluetooth
Description This class provides the primary interface for discovering and establishing a Bluetooth network connection.
Fields
EVENT_DISCOVER_DEVICE   Reported when a nearby device is discovered.
EVENT_DISCOVER_DEVICE_COMPLETED   Reported when the device discover process has completed.
EVENT_DISCOVER_SERVICE   Reported when a service is discovered on a device.
EVENT_DISCOVER_SERVICE_COMPLETED   Reported when the service discover process has completed.
EVENT_CLIENT_CONNECTED   Reported when a device has connected to this service.
Methods
cancel()   Cancels a search for devices.
discover()   Searches for nearby Bluetooth devices.
find()   Searches for nearby Bluetooth devices running a specific service.
start()   Starts a Bluetooth service that other devices can connect to.
stop()   Stops sharing a Bluetooth service.
Constructors 
Bluetooth(parent)
Bluetooth(parent, uuid)
Parameters
parent   PMIDlet: typically use "this"
uuid   String: a 32-character hexadecimal value representing a 128-bit universally unique identifier for this service

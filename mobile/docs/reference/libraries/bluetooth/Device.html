Bluetooth library
Name

Device
Description Objects of this class represent nearby devices discovered on the Bluetooth network.
Fields
name   The name of the device. The name may be (Unknown) until the discovery process is complete.
address   A string uniquely identifying the device on the network.
Methods
cancel()   Cancels a search for services.
discover()   Searches for services on the device.

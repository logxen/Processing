Capture class
Name

show()
Description Displays the viewfinder output of live video from the camera. This output will overlay on top of the sketch. Note that, if you are drawing frames of animation while the viewfinder is showing, you may overwrite the viewfinder and cause flickering on the display. It is recommended that you do not draw to the screen while the viewfinder is showing by calling noLoop() to stop the main animation loop.

You may optionally specify a position and size for the viewfinder, although not all phones will allow the the viewfinder to be changed. The actual position and size of the viewfinder is reflected in the viewX, viewY, viewWidth, and viewHeight variables in the Capture class.
Syntax 
cap.show()
cap.show(x, y)
cap.show(x, y, width, height)
Parameters
cap   Capture: any variable of type Capture
x   int: x coordinate of upper left corner of viewfinder
y   int: y coordinate of upper left corner of viewfinder
width   int: width of viewfinder
height   int: height of viewfinder
Returns void

PImage2 class
Name

loadPixels()
Description Copies the pixel data for the output window into the pixels[] array of the image. Anything that has been drawn to the output so far in draw() will be copied into the image. If the PImage2 object is not big enough for the entire screen, the image will be cropped. Otherwise, use the optional parameters to specify exactly the area you wish to copy from and its destination in the PImage2 object.
Syntax 
img.loadPixels()
img.loadPixels(sx, sy, swidth, sheight, dx, dy, dwidth, dheight);
Parameters
img   any variable of type PImage2
sx   int: X coordinate of the source's upper left corner
sy   int: Y coordinate of the source's upper left corner
swidth   int: source image width
sheight   int: source image height
dx   int: X coordinate of the destination's upper left corner
dy   int: Y coordinate of the destination's upper left corner
dwidth   int: destination image width
dheight   int: destination image height

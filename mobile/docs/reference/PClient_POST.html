PClient class
Name

POST()
Description The POST method is used to submit data to a web server to be processed. This is the method most commonly used by web browsers to send data from forms when the user clicks on a "Submit" button. The params array contains the names of the form fields. The values array must be the same size as and contain the values corresponding, in order, to the names specified in the params array. If you need to submit binary data, create an array of Objects that can contain either String values or byte array (byte[]) objects containing the binary data.
Syntax 
POST(file, params, values);
POST(file, params, data);
Parameters
file   String: name of the web page/script that will process the data
params   String[]: an array of String objects containing the names of form fields being submitted
values   String[]: an array of String objects containing the values of form fields being submitted
data   Object[]: an array of objects, each object either String or byte[], representing values or binary data being submitted
Returns PRequest

Name

loadFont()
Examples 
PFont font;
// The font must be located in the sketch's 
// "data" directory to load successfully
font = loadFont("FFScala-32.mvlw"); 
textFont(font); 
text("word", 15, 50);

Description Loads a font into a variable of type PFont. Bitmap fonts can be created by selecting Create Font... from the Tools menu. To load correctly, bitmap fonts must be located in the data directory of the current sketch. Note that bitmap fonts created with Mobile Processing are not compatible with bitmap fonts created with Processing and vice versa.

Bitmap fonts can consume a large amount of memory. Mobile Processing also allows you to load system fonts from the mobile phone. Use the FACE, STYLE, and SIZE constants to specify a system font. Not all phones support all combinations of FACE, STYLE, and SIZE, but will return the closest matching font.
Syntax 
loadFont(fontname)
loadFont(fontname, color)
loadFont(fontname, color, bgcolor)
loadFont(face, style, size)
Parameters
fontname   string: name of the font to load
color   color: color of the font
bgcolor   color: background color to anti-alias font with
face   int: one of FACE_SYSTEM, FACE_MONOSPACE, or FACE_PROPORTIONAL
style   int: one of STYLE_PLAIN, STYLE_BOLD, STYLE_ITALIC, or STYLE_UNDERLINED
size   int: one of SIZE_SMALL, SIZE_MEDIUM, or SIZE_LARGE
Returns PFont

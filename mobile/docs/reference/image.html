Name

image()
Examples 
PImage b;
// Images must be in the "data" directory to load correctly
b = loadImage("laDefense.jpg");
image(b, 0, 0);

Description Diplays images to the screen. The images must be in the sketch's "data" directory to load correctly. Select "Add file..." from the "Sketch" menu to add the image. Mobile Processing currently works with .png images. If an image has transparency, it will maintain its transparency. However, support for alpha depends upon the target device. The img parameter specifies the image to display and the x and y parameters define the location of the image from its upper-left corner.

To display only a sub-area of the image, specify the upper-left corner and dimensions of the sub-area. The imageMode() function changes the way the parameters work. For example, a call to imageMode(CORNERS) will change the width and height parameters to define the x and y values of the opposite corner of the image.
Syntax 
image(img, x, y)
image(img, sx, sy, swidth, sheight, x, y)
Parameters
img   PImage: the image to display
x   int: destination x-coordinate of the image
y   int: destination y-coordinate of the image
sx   int: x-coordinate of the sub-area in the image
sy   int: y-coordinate of the sub-area in the image
swidth   int: width of the sub-area in the image
sheight   int: height of the sub-area in the image

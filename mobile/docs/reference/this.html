Examples 
int ypos = 50;

void setup() {
  size(100, 100);
  noLoop();
}

void draw() {
  line(0, 0, 100, ypos);
  this.ypos = 100;
  line(0, 0, 100, ypos);
}
 

import processing.video.*;
Capture cap;

void setup() {
  cap = new Capture(this);
  cap.show(0, 0, 176, 144);
}

Description Refers to the current object (i.e. "this object"). In Processing, it's most common to use this to pass a reference from the current object into one of the libraries.
